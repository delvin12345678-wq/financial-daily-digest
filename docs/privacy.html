隱私權政策

我們如何處理你的資料

短版:你的 Email 用來寄日報,你的股票偏好用來客製日報內容,LINE userId 用來推即時提醒。我們不賣資料、不共享給第三方廣告商、不追蹤你瀏覽其他網站。

最後更新:2026-05-23

1. 我們蒐集哪些資料

當你使用 MarketDaily(marketdaily.ai),我們會蒐集以下資料:

  • 必要資料:Email、密碼(雜湊儲存,我們看不到原文)、訂閱方案(免費 / Pro / Premium)、訂閱起始時間
  • 個人化用資料:你在後台設定的美股 / 台股股票偏好
  • 選用資料:LINE userId(只有你主動「綁定 LINE」時才取得,用於推送即時提醒)
  • 系統資料:來源 UTM 標籤、頁面點擊、註冊裝置粗略地理位置(僅國家層級)
  • 金流資料:付費訂閱透過 Stripe 處理,我們不接觸你的信用卡資料,只收到 Stripe 回傳的訂閱狀態

2. 我們為什麼蒐集

  • Email:寄送你訂閱的每日財經日報、帳號通知、付款收據
  • 股票偏好:讓日報只分析你關心的個股,不是給所有人一樣的內容
  • LINE userId:當重大新聞影響你的持股時即時推播(Premium 功能,需你主動綁定才會用到)
  • UTM / 行銷分析:了解哪個來源的訂戶最多,優化我們的內容

3. 我們不做這些事

  • 不販售任何個資給第三方
  • 不分享給廣告商或數據經紀
  • 不放第三方追蹤碼(沒有 Facebook Pixel、沒有 Google Analytics 追蹤)
  • 不在 Email 中夾帶與你訂閱主題無關的廣告

4. 資料儲存在哪

所有資料儲存在 Cloudflare KV(全球 CDN 邊緣節點)與 Brevo(歐盟 GDPR 合規 Email 服務)。台灣與美國用戶資料可能存於不同邊緣節點以加快回應速度。

5. 你有什麼權利

  • 取得備份:寄信到下方聯絡 Email,我們會把你所有資料以 JSON 格式寄給你
  • 修改 / 更正:登入 dashboard.html 直接改,或寄信請主編改
  • 刪除帳號:任何一封日報底部點「取消訂閱」即時生效;若要連同所有 KV 紀錄一併清除,寄信給我們會 7 日內完成
  • 退費:30 天無理由退費保證

6. Cookie / 本機儲存

MarketDaily 使用 localStorage / sessionStorage(不是傳統 cookie)儲存:你的登入 Email、語言偏好、UTM 來源標籤、推薦碼。這些資料只存在你的瀏覽器,我們不會主動讀取,也不會跨網站追蹤。

7. LINE 綁定(選用)

當你在 dashboard 點「綁定 LINE」,我們會帶你到 LINE Login OAuth 頁面。授權後我們只取得你的 LINE userId(一串匿名字串),不取得你的 LINE 好友清單、訊息內容、Profile 照片。userId 僅用於 LINE Bot 推播即時提醒給你個人。你可在 dashboard「解除綁定」隨時撤銷。

8. 兒少保護

本服務不對 13 歲以下用戶提供服務。若你發現有未滿 13 歲用戶提供個資給我們,請通知主編,我們會在 48 小時內刪除其資料。

9. 政策變更

政策若有重大變更,我們會透過你訂閱的 Email 發送通知,並在此頁更新「最後更新」日期。

10. 聯絡我們

有任何隱私權相關問題,寄信:marketdailyhq@gmail.com,或從 聯絡頁 送問題單,主編 24 小時內回覆。
