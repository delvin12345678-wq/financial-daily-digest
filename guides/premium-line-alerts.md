# MarketDaily Premium ⚡ 即時 LINE 重大新聞提醒｜使用說明

> Premium 專屬功能。當美國或其他市場出現**會影響你持股的重大新聞**時,我們在數分鐘內用 LINE 直接通知你 —— 不必等隔天早上 7 點的日報。

---

## 這個功能在做什麼

系統每 2 分鐘掃描財經新聞,自動比對你在後台設定的持股。一旦某則新聞:

1. 命中重大事件(財報、併購、財測下修、監管調查、訴訟、停牌、評等調整…),**且**
2. 跟你持有的個股相關,**且**
3. 經 AI 判定嚴重度達 7/10 以上,

就會即時推一則 LINE 訊息給你。一般、不影響你持股的新聞不會打擾你。

## 你會收到什麼

提醒是一則 LINE 文字訊息,長這樣:

```
🚨 重大消息｜你的持股 輝達 Nvidia

Nvidia 預警下季財測,資料中心需求轉弱

💡 為什麼跟你有關
財測下修通常直接壓低股價,留意盤前反應。

🔗 原文:https://...
—— MarketDaily 即時提醒｜僅供參考,非投資建議
```

嚴重度特別高(9–10 分)時,標題會多一個「⚠️ 高度重大」標記。

## 幾個規則

| 規則 | 說明 |
|------|------|
| 每日上限 | 每人每天最多 5 則,避免訊息轟炸。 |
| 不重複 | 同一個事件就算多家媒體報導,你只會收到 1 則。 |
| 半夜照推 | 美股盤中常是台灣深夜 —— 重大消息不分時段照推,確保你不錯過。 |
| 僅供參考 | 所有提醒都是資訊整理,**不是投資建議**,進出請自行判斷。 |

## 如何解除綁定

後台同一張「即時 LINE 重大新聞提醒」卡片 →「**解除綁定**」。
解除後不再收到提醒,可隨時重新綁定。

## 常見問題

**Q:沒收到提醒,正常嗎?**
多數時候是因為當下沒有達標的重大新聞 —— 這是好事,代表你的持股沒出大事。若想確認系統有在運作,可檢查後台卡片是否顯示「已綁定」。

**Q:我加了好友也綁定了,但完全收不到。**
依序檢查:① 確認用 LINE 點過上面的加好友連結;② 確認帳號是 Premium 方案;③ 後台卡片狀態是否為「已綁定」。若卡片提示「請重新綁定」,代表綁定已失效,點一次重新綁定即可。

**Q:可以只接收某幾檔股票的提醒嗎?**
提醒範圍就是你在[股票偏好](https://marketdaily.ai/preferences.html)設定的持股。想縮小範圍,調整偏好清單即可。

**Q:提醒和每日日報有什麼不同?**
日報是每天早上 7 點的完整個人化彙整;即時提醒只在「重大且影響你持股」時觸發,主打速度 —— 兩者互補。
